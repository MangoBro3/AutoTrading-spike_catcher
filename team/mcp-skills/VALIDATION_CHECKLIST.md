## E. Go / No-Go
- Go 조건
  - [ ] 공통 게이트 100% 충족
  - [ ] 역할별 필수항목 미해결 0건
- No-Go 조건
  - [ ] 권한 위반 1건 이상
  - [ ] 테스트 미통과 병합 시도 존재
  - [ ] 문서-코드 불일치 치명 이슈 존재
